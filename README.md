## 📌 Project Overview

This project performs real-time object detection and tracking using the YOLOv8 model and OpenCV. The system captures live video from a webcam, identifies multiple objects in real time, tracks them across frames, and displays annotated results with bounding boxes and labels.

The project demonstrates practical applications of Computer Vision, Deep Learning, and Object Tracking.

---

## 🚀 Features

✅ Real-time object detection

✅ Multi-object tracking

✅ Bounding box visualization

✅ Confidence threshold filtering

✅ Webcam integration

✅ Fast inference with YOLOv8

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Madhura-Malap/real-time-object-tracking-yolov8.git
```

Install dependencies:

```bash
pip install ultralytics opencv-python
```

---

## ▶️ Run the Project

```bash
python object_detection.py
```

Press **Q** to exit.

---

## 🎥 Working

1. Access webcam feed.
2. Load YOLOv8 model.
3. Detect objects in each frame.
4. Track detected objects.
5. Display annotated video output.
